<h1 align="center"><strong>Hariboot</strong></h1>

<br>
**Hariboot is the UEFI bootloader of the HaribOS operating system.**

## To Install
### debian/ubuntu
- clang
- mtools
- xorriso
- lld
- make
- qemu-system

# Compiling

In a shell, run the command above:
```
make hariboot
```

# Run the bootloader

To run the bootloader, just executable the following command:
```
sudo make run
```
